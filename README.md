# ClashNodes

自动抓取合并互联网上的公开节点。修正 Base64 格式，用于各种 Clash, V2ray, sing-box 等订阅转换。

## 🚀 订阅链接

为了方便直接使用，以下链接均已通过 `ghproxy.net` 进行加速。

### 1. Base64 订阅地址
适用于 V2Ray, Shadowsocks, Trojan 等客户端。
- [点击复制链接](http://ghproxy.net/https://raw.githubusercontent.com/Yaka88/ClashNodes/master/list.txt)
- `http://ghproxy.net/https://raw.githubusercontent.com/Yaka88/ClashNodes/master/list.txt`

### 2. Clash 订阅地址
适用于 Clash, Clash for Windows, Clash Meta 等客户端。
- [点击复制链接](http://ghproxy.net/https://raw.githubusercontent.com/Yaka88/ClashNodes/master/list.yml)
- `http://ghproxy.net/https://raw.githubusercontent.com/Yaka88/ClashNodes/master/list.yml`

### 3. Clash Meta (Mihomo) 订阅地址
包含更多元数据支持的 Clash 订阅。
- [点击复制链接](http://ghproxy.net/https://raw.githubusercontent.com/Yaka88/ClashNodes/master/list.meta.yml)
- `http://ghproxy.net/https://raw.githubusercontent.com/Yaka88/ClashNodes/master/list.meta.yml`

## 🛠 功能说明
- **自动抓取**: 定时从互联网公开源获取节点信息。
- **格式修正**: 自动修复 Base64 编码错误。
- **多格式支持**: 提供通用 Base64 及 Clash 专用 YAML 格式。
